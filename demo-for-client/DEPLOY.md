# Deploy to Internet (Free Options)

## Option 1: Railway (Easiest)
1. Push code to GitHub
2. Connect Railway to GitHub
3. Deploy automatically
4. Get URL like: `https://yourapp.railway.app`

## Option 2: Render
1. Upload simple-backend folder
2. Set start command: `npm start`
3. Get free URL

## Option 3: Heroku (Classic)
1. Install Heroku CLI
2. `heroku create yourrestaurant`
3. `git push heroku main`

## Cost: $0/month for small restaurants

## What Customers Get:
- Reserve from anywhere: `https://yourrestaurant.railway.app`
- Staff access same URL
- Data persists in JSON files on server

## Production Ready:
- Add custom domain: `yourrestaurant.com`
- SSL certificate (free with hosting)
- Backup JSON files regularly
